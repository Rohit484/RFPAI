## Getting Started

1. **Clone the Repository:**
```b
git clone 
```

2. **Install Dependencies:**
```bash
pip install -r requirements.txt
```

3. **Run the App:**
```bash
uvicorn main:app --reload
```

4. **Check documentation**
Access `localhost:port/docs`
